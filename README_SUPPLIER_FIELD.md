# เพิ่มเติม — ฟิลด์ "บริษัทที่ซื้อ" (Supplier Company)

ทำเสร็จแล้วครับ ตามที่ค้างไว้จากตอนทำ Phase 4-5

## สิ่งที่เปลี่ยน

- เพิ่มคอลัมน์ `supplier_company` (free text) ในตาราง `vehicles` — migration ใหม่: `supabase/migrations/0005_vehicle_supplier_company.sql`
- พิมพ์รหัสบริษัทเองได้อิสระตามที่ใช้อยู่แล้ว เช่น `TAC`, `TV-`, `TM-`, `TS-` หรืออะไรก็ได้ ไม่ล็อกเป็น dropdown ตายตัว
- หน้า **เพิ่ม/แก้ไขรถยนต์** มีช่องกรอก "บริษัทที่ซื้อ" เพิ่มแล้ว
- หน้า **คลังรถยนต์**:
  - มี **dropdown กรองตามบริษัทที่ซื้อ** (จะแสดงเฉพาะตอนมีข้อมูลอย่างน้อย 1 บริษัทในระบบ) — นี่คือ "filter" ที่ถามไว้ตอนแรก ไม่ต้องแยกชีตแล้ว
  - ช่องค้นหาก็ค้นรหัสบริษัทได้ด้วย
  - แสดง badge เล็กๆ ใต้รหัสสต็อกในตารางให้เห็นบริษัทเร็วๆ

## ขั้นตอนอัปเดตฐานข้อมูล

```bash
supabase db push   # จะรัน 0005_vehicle_supplier_company.sql
```

ข้อมูลรถเดิมที่มีอยู่แล้วจะมีค่า `supplier_company` เป็น NULL ไปก่อน — เข้าไปแก้ไขทีละคันเพื่อใส่รหัสบริษัทย้อนหลังได้ตามสะดวก หรือถ้ามีข้อมูลเดิมในชีตเก่าจำนวนมาก บอกได้ครับ ผมช่วยเขียนสคริปต์ import จาก CSV ให้

ตอนนี้พร้อมไปต่อ **Phase 9: แดชบอร์ดฉบับสมบูรณ์** แล้วครับ พิมพ์ "continue" ได้เลย
